# Roadmap

## Done
- Bundled readable fonts (OpenDyslexic, Lexend, Atkinson Hyperlegible) plus the
  system font and any device-installed font.
- Text size, line height, letter spacing, word spacing.
- Defaults to per-site cascade, with each setting showing its source (Default /
  This site), inline override and reset.
- On by default, per-site disable, Undo, and a "View all customizations" manager.
- Onboarding / help page. macOS and iOS targets. Local-only, no remote loading.

## Next
- Readable presets (Dyslexia, Low Vision, Calm Reading, Code-friendly, Original).
- Background tint, link colour, and contrast (these unlock the richer presets).
- Max text width and text alignment for long-form articles.
- Per-change scope picker (this page / this site / defaults) and page-level rules.
- Export / import settings as JSON; search customizations.
- Keyboard shortcut to toggle the current site.
- iPad support. The extension already works on iPad Safari; the first release
  ships iPhone-only just to get it out sooner, and to skip the multi-gigabyte
  iPad simulator install. Re-enabling it is a device-family flip plus iPad
  screenshots.

## Maybe later
- iCloud sync across your own devices (no accounts).

## Out of scope (on purpose)
- AI rewriting, text-to-speech, OCR, reading overlays, subscriptions, and any
  cloud font fetching.
