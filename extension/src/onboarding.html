Font Changer

Make any web page easier to read. Pick a font, size, and spacing that work for you, on by default everywhere. Open source, and private by design.

Turn it on for every site

Safari asks you to grant access once. On your Mac:

  1. Open Safari → Settings (⌘,) → Extensions.
  2. Tick Font Changer to enable it.
  3. Select it and set its website permission to Allow on Every Website → Always Allow.

Running an unsigned dev build? If it ever vanishes from the list, re-enable Develop → Allow Unsigned Extensions (it resets when Safari restarts).

On iPhone or iPad

  1. Settings app → Apps → Safari → Extensions.
  2. Tap Font Changer and turn it on.
  3. Set Other Websites to Allow so it works everywhere.

Your privacy

  • No tracking, no analytics, no accounts.
  • Fonts are bundled in the app. Nothing is loaded from the internet.
  • Your settings stay on this device.
  • The extension only restyles pages. It never reads or sends your browsing.

Adjusting it later

Click the Font Changer button in the Safari toolbar any time to switch the font, change text size, line height, letter or word spacing, or opt the current site out. Pick a value by dragging the slider or typing it. Your choices are remembered on this device.
